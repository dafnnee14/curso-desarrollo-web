Definir estilos del HTML con CSS

CSS (cascade Style Sheets) se una para controlar el formato visual de una página.

con CSS se puede controlar el color, fuente, tamaño del texto, espaciado, alineamiento, imagenes de fondo, como se ve una pagina dependiendo del tamaño de pantalla del dispositivo que estemos usando, etc.

Formas de usar CSS

  • Inline - Usando el atributo <style> dentro del elemento HTML
  • Internal - Usando la etiqueta <style> dentro de la seccion del <head>
  • External - Usando un archivo CSS externo

Inline o CSS interno

se usa para aplicar un estilo a un elemnto HTML en concreto mediante el atributo <style> dentro de la etiqueta del elemento. Ejemplo de un titulo con la letra azul:

Internal CSS

Se usa para aplicar estilos a un documento en concreto y se define dentro de la etiqueta <style> dentro de la sección <head> del documento. Referencia en al linea 9 del codigo del documento.

External CSS

Es un archivo con la extension ".CSS" que contiene exclusivamente codigo CSS. Se puede aplicar al documento mediante el elemento <link>. Referencia en la linea 8.

Las dos principales ventajas de externalizar el CSS son:

  • Los estilos se pueden aplicar a tantos documentos como querramos.
  • La longitud del código disminuye sensiblemente.
