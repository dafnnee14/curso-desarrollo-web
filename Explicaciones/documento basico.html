Formateo del texto en HTML

Elementos destinados especificamente a Formatear el texto:

  • b - texto en negrita
  • strong - texto importante
  • i - texto cursiva
  • em - texto énfasis
  • mark - texto marcado
  • small - texto de menor tamaño
  • del - texto tachado
  • ins - texto insertado
  • sub - texto subescrito
  • sup - texto superescrito

Esto es un párrafo. El texto que querramos destacar visualmente lo haremos con negrita, el texto importante irá con la etiqueta strong.
mientras que la b es meramente visual, strong influye en el posicionamiento web de la pagina.

Elementos de citación

  • blockquote - Define una seccion de otra fuente.
  • q - Define una cita corta.
  • abbr - Define una abreviatura o acronimo.
  • address - Define la información de contacto del autor del articulo.
  • cite - Define el titulo de un trabajo creativo. Como un poema, libro o canción.
  • bdo - Se usa para definir la dirección del texto.

Estilos

Los estilos definen el aspecto visual de los elementos HTML que acompañan. La sintaxis que es la siguiente:
