Animaciones con CSS

CSS permite animar elementos HTML sin necesidad de recurrir a JavaScript.

Qué son las animaciones?

Una animación es el cambio gradual de un estilo a otro.

Se pueden cambiar tantas propiedades de CSS como se quiera, tantas veces como sea necesario.

Para usar una animación debe especificarse la regla @keyframes

La regla @keyframes

Esta regla contiene los estilos que el elemento tendrá en determinados momentos.

Cuando se especifica una regla, los cambios se ejecutarán de manera gradual en ciertos momentos.

Para que la animación funcione, debes asignar un keyframe al CSS de un elemento.

Propiedades de las animaciones

  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode

animation-delay:

Especifica el retardo del inicio de una animación.

Los valores positivos indican el tiempo que ha de transcurrir para la animación de comienzo. Los valores negativos indican que la animación empezará como si ya hubiese transcurrido el tiempo especificado.

animation-iteration-count

Establece el número de veces que se repetirá una animación.

Se puede establecer un número natural de veces o se puede indicar un número ilimitado con el valor infinite.

animation-direction

Especifica si la animación se producirá en orden normal, inverso o irá alternando ciclos. Los valores posibles son:

  • normal La animación se reproduce hacia adelante. Valor por defecto.
  • reverse La animación se reproduce hacia atrás (de 100% a 0%).
  • alternate La animación se reproduce hacia adelante, y luego hacia atrás.
  • alternate-reverse La animación se reproduce hacia atrás, y luego hacia adelante.

animation-timing-function

Especifica la curva de velocidad al reproducir una animación.

Puede tener los siguientes valores:

  • ease La animación tendrá un inicio lento, luego irá rápido y tendrá un final lento. Valor por defecto.
  • linear La animación tendrá una velocidad estática durante toda la duración.
  • ease-in La animación tendrá un inicio lento.
  • ease-out La animación tendrá un final lento.
  • ease-in-out La animación tendrá un inicio lento y un final lento.
  • cubic-bezier(n, n, n, n) Permite al usuario especificar la curva de velocidad mediante una función.

animation-fill-mode

Por defecto las animaciones no afectan a los elementos antes de iniciarse o despues de finalizar. Esta propiedad cambia este comportamiento.

La propiedad animation-fill-mode establece el estilo del elemento objetivo cuando la animación no se esta reproduciendo. (antes de empezar, despues de finalizar o ambas).

Los valores posibles:

  • noneValor por defecto. La animación no aplicará cambios al estilo.
  • fordwards El elemento retendrá el estulo establecido en el último keyframe de la animación (dependerá de si se esta reproduciendo normal o de manera invetida animation-iteration-count).
  • backwards El elemento retendrá el estilo establecido en el primer keyframe de la animación. Dependerá de la dirección y tomará el estilo durante el retardo proporcionado por animation-delay
  • both La animación seguirá las reglas combinadas de fordwards y backwards.
