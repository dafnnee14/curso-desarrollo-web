"Layout" en CSS

El layout es la disposición o la forma en la que los elementos se muestran en pantalla según su tipo.

Display

Es la propiedad más importante de CSS para controlar el alineamiento.

La propiedad display especifica la forma en la que se muestra un elemento.

En HTML cada elemento posee un valor por defecto dependiento del tipo de elemento que sea. La mayoría de los elementos poseen una propiedad block o inline por defecto.

Elementos de bloque

Los elementos de bloque siempre empiezan en una nueva línea y toman todo el ancho disponible.

Este elemento div es de tipo bloque.

Algunos elementos de bloque:

  • h1 - h6
  • p
  • form
  • header
  • section
  • footer

Elementos en línea (Inline)

Estos elementos no empiezan en una nueva línea y ocupan sólo el ancho necesario.

Esto es un span dentro un párrafo.

Algunos de estos elementos:

  • span
  • a
  • img

Diferencias entre display: none y visibility: hidden

Al ocultar un elemento con display: none la página se mostrará como si el elemento no es estuviera ahí.

Al ocultar un elemento con visibility: hidden el elemento no se mostrará pero el espacio que ocupa quedará en blanco.

width y max-width

Como hemos mencionado anteriormente, un elemento de bloque siempre ocupa todo el ancho disponible.

Asignar una anchura con width evitará que un elemento de bloque se estire a los bordes de su contenedor. Entonces podremos asignar su margen margin: auto para centrar horizontalmente el elemento respecto a su contenedor. El elemento pasará a ocupar el ancho especificado y el resto del espacio sobrante será repartido equitativamente entre sus márgenes.

El problema de esto, es que al asignar un espacio fijo nos dará problemas en pantallas pequeñas. El elemento se sladrá del contenedor y el navegador añadirá una barra de scroll horizontal

Si por el contrario, usamos max-width, mejorará el manejo del navegador de estos elementos. Esto es muy importante a la hora de contruir sitios web para dispositivos pequeños.

La propiedad position

Esta propiedad define el método de posicionamiento de un elemento.

Hay 5 tipos de posición.

  • static
  • relative
  • fixed
  • absolute
  • sticky
    •

Los elementosse posicionan con las propiedades top, left, right, bottom. Sin embargo estas propiedades no funcionarán sin especificar primero la position. Además, dependiendo del tipo de position las propiedades funcionarán de manera distinta.

position: static

Todos los elementos tienen este valor por defecto.

Las propiedades top, bottom, left y right no afectan a los elementos con posición estática.

position: relative

Un elemento con este valor se posicionará de manera relativa a su posición normal. Es decir, el centro del eje será sobre sí mismo.

Este div tiene position: relative

position: fixed

Un elemento con este valor, se posicionará relativamente respecto a la pantalla. Lo que quiere decir que tendrá siempre la misma posición incluso si scrolleamos la web.

position: absolute

Un elemento con este valor se posicionará respecto a su contenedor posicionado (no estático).

Si un elemento absoluto no tiene ancestro posicionados, usará el body y actuará como un position: fixed.

Este elemento div tiene una position: relative
Este elemento div tiene una position: absolute

position: sticky

Un elemento sticky alternará su comportamiento entre relative y fixed dependiendo de la posición de scroll de la página.
