Grid o rejilla

El sistema grid de CSS permite utilizar una plantilla de rejilla, con filas y columnas, para alinear de manera sencilla páginas web sin necesidad de usar float o posicionamiento.

Elementos grid

Una rejilla consiste en un elemento padre, con uno o más hijos.

El elemento padre debe de tener la propiedad display: grid o inline-grid. Los elementos descendientes se convertirán automáticamente en objetos(items) grid.
