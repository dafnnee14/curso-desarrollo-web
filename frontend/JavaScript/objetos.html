Objetos en JavaScript

Un objeto es un especial de variable, que almacena distintos valores clasificados como propiedades.

Un objeto también puedes almacenar distintos métodos. Éstos son bloques de código que se ejecutarán en determinados momentos,

En la vida real, un coche es un objeto.

Un coche tiene propiedades como peso y color, y métodos como arrancar y parar.

Todos los coches tienen las mismas propiedades, pero éstas tienen un valor distinto dependiendo del coche.

Todos los coches tienen las mismos. métodos, pero éstos se pueden ejecutar en distintos momentos.

Normalmente, usamos una variable para guardar un valor sencillo:

let coche = "Renault"

Los objetos son variables que pueden contener múltiples valores: 

        
            const coche = {
                marca:"Renault"
                Modelo:"Megane"
                color:"Gris"
            }

Los valores se escriben emparejados con el nombre de la propiedades name: value.
