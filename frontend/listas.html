Listas de HTML

Listas sin orden

Se usa la etiqueta <ul> (unordered list) como contenedor. Cada elemento de la lista irá en una etiqueta <li>:

Se usan para mostrar listas en las que el orden de los elementos no es importante. Por ejemplo: La lista de la compra, de asistencia, de material escolar.

  • Fruta
  • Pasta
  • Carne
  • Leche

Listas ordenadas

Se usa la etiqueta <ol> (ordered list) como contenedor. Cada elemento de la lista irá en una etiqueta <li>:

Son listas numeradas. Se usan cuando el orden es importante y los elementos tienen importancia según el orden o son pasos a seguir, por ejemplo.

Algunos ejemplos de listas ordenadas son: Una receta, un manual/libro que va ordenado por capítulos (la lista sería el índice)...

Receta del café con leche

  1. Preparar el café
  2. Añadir la leche caliente
  3. Añadir azúcar al gusto
  4. Servir

Listas descriptivas

Son listas de términos, cada uno de éstos con su propia descripción. Se usa la etiqueta <dl> (description list) como contenedor, cada término irá en un <dt> y cada descripción en un <dd>

Café
- Bebida de color negro que se bebe caliente.
Leche
- Bebida de color blanco que se bebe fría o caliente.
