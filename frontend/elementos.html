Links o Hipervinculos

Los links se encuentran en todo tipo de paginas, permiten al usuario navegar por la web o por los documentos de la web.

Cuando clicas en un link te lleva a otro documento o a una parte concreta del mismo.

Cuando se pasa el cursor sobre el link, éste se torna en mano.

un link se define mediante el elemento <a>

La sintaxis de un link por norma general es:
<a href="url"> Texto del link</a>

El atributo más importante de un link es el "href" ya que es éste el que indica el punto al que nos llevara el link.

Los link tienen 3 estados:

  • Son azules cuando no han sido visitados.
  • Morados cuando ya han sido clicleados.
  • Rojos cuando están activos.

Los link se pueden modificar mediante CSS para que tengan un asprecto completamente dsitinto.
