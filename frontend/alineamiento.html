Usando flexbox

Para alinear elementos uando flexbox. Éstos deben de estar ubicados dentro de un elemento padre al que vamos a llamrar contenedor con la propiedad display: flex:

Los elementos hijos ( que esten dentro) de un contenedor flex se convertirán automáticamente, en elementos flexibles.
