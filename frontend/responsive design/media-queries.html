Media Queries

En CCS3, se introdujo la idea de que en vez adaptar las webs según el tipo de dispositivo desde el que se está visualizando, las adaptaremos según las capacidades.

Esto quiere decir que en vez revisar si el dispositivo es una pantalla, un móvil, una impresora, etc, etc... Simplemente nos basaremos en los siguientes parámetros:

  • ancho y alto del viewport
  • ancho y alto de la pantalla del dispositivo
  • orientación - si el dispositivo está en vertical u horizontal.
  • resolución

Usar media queries nos capacita con la posibilidad de hacer páginas hechas a medida para pantallas de escritorio, portátil, tablets, móviles... etc.

Sintaxis

Una expresión de media query puede contener una o varias condiciones que se resuelven en true o false 

            @media not|only mediatype and (condiciones){
                Código CSS
            }
